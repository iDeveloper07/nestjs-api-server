## Description

NestJs API server

## Requirment

1. Install Node
2. Install MongoDB 
3. Install RabbitMQ  

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
